## 2.1 数据类型分类

Java的数据类型包括两类：

- 基本数据类型：一些较为简单的数据类型，主要包括：整型，浮点，字符，字符串，布尔类型
- 引用数据类型：由地址指向实例的数据类型，变量保存实例的地址，我们通过地址(引用)，间接地访问到实例

## 2.2 命名

Java 的变量名称(标识符)需要符合特殊规则：

- 由字符，数字，下划线组成
- 不能以数字开头
- 不与 Java 保留字重复

## 2.3 基本类型

整型就是整数, Java中整型一共有四种

- int：4个字节，表示范围在$2^-32~2^(32-1)$
- long：8个字节
- short： 2个字节
- byte： 1个字节

```java
int a = 1;
long b = 2;
short c = 3;
byte c = 4; 

```

运行时，如果变量数值大小超过它可以表示的范围，就会**溢出**

---

字符类型表示一个字符, 用一对单引号括起来的单个字符就是字符类型

```java
char ch = 'a'
```


---

浮点，就是包含小数点的数字，包括单精度浮点 float 和双精度浮点 double。 float 长度为32字节，表示范围为 $-3.4E^38~3.4E^38$, double 长度为64字节， 表示范围为 $ -1.7E^308~1.7E^308$.

```java

float f = 3.14;
double d = 3.1415926;
```

Java默认的浮点型为 double ,即 `3.14` 的类型是 double，我们可以在它的后面加上 d 或 f 表示类型。

```java
3.14 // double
3.14d // double
3.14f // float
```

---

bool 类型用于表示真假， 由两个值 true， false

## 2.4 类型转换

我们可以把数据类型进行强制转换

```java
int a = (int) 3.14; // 把3.14 double为int 舍去小数， a =1
System.out.println(a); // 1

```


有些数据类型会自动进行类型转换

```java
float a = 3.14 // 3.14 double, 赋值给 float 变量是， 自动转换为 float
```


## 2.4 引用类型

引用类型是一种通过地址访问实例的类型, 保存统一地址的变量操作会作用于实例，同步体现在另一个变量中。

字符串类型是引用类型，它的值用一对双引号括起来。

```java
String str = "It is a String";
// 我们将 str 保存的字符赋值给另一个变量
String str1 = str;
str1 = "another"
System.out.println(str); //输出 another
```

---



数组是引用类型，表示 一组同类型的数据集合。

```java
int[] arr = {1, 2, 3, 4}
```

除了上面变量定义时初始化，我们还可以使用 new 关键字获取数组

```java
int[] arr = null;
arr = new int[4]
arr[0] = 1;
arr[1] = 2;
arr[2] = 3;
arr[3] = 4;

```
