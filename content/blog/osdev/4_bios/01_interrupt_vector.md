## 软盘中断

### 软盘读, 写, 校验, 寻道

| 中断号 | 参数                                                                                                                                                                                                        | 返回值                                                                 | 说明 |
| ------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- | ---- |
| 0x13   | AH=0x02(读盘),<br />AL=读取的扇区数目,<br />CH=起始柱面号&0xFF,<br />CL=起始扇区号(0-5位)  ，起始柱面号&0x300>>2<br />DH=起始磁头号,<br />DL=驱动器号(0x0表示A驱动器, 0x1表示B驱动器),<br />ES:BX=目的地址 | FLAGS.CF=0, 没有错误，AH=0.<br />或<br />FLAGS.CF=1, 有错误，AH=错误号 |      |
| 0x13   | AH=0x03(写盘)                                                                                                                                                                                               |                                                                        |      |
| 0x13   | AH=0x04(校验)                                                                                                                                                                                               |                                                                        |      |
| 0x13   | AH=0x0c(寻道)                                                                                                                                                                                               |                                                                        |      |
